CSS Color Cheat Sheet

Backround, Color, and Linear Gradient

Let's learn to make cool colors faster!

Background Color Element

  • In styles.css code background-color: color;
  • This will apply a background color to your selected element.

Color Element

  • In styles.css code color: color;
  • This will apply a foreground color to your selected element.

Linear Gradient

  • In styles.css code background-image: linear-gradient(num+deg e.g. 180deg, color1, color2);
  • This will apply a linear color to your selected element and 180 degrees.
Lets see a couple of examples below:
Topic CSS rule Examples
Background-Color h1 {
background-color: green;
}
Color h1 {
color: green;
}		 Green
Linear-Gradient h1 {
background-image: linear-gradient(90deg, OliveDrab, SeaGreen);
}
